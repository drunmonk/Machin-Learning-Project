# Micrograd from Scratch

This repository contains my personal implementation of **Micrograd**, inspired by the video lectures of **Andrej Karpathy**.

Micrograd is a tiny **autograd engine** with a simple neural network library built from scratch in Python.  
This project is part of my learning journey to understand:

- Backpropagation
- Computational graphs
- Automatic differentiation
- Neural networks from first principles

## Features
- Scalar-valued autograd engine
- Forward pass computation
- Backward pass / gradient calculation
- Basic neural network components

## Inspiration
Based on Andrej Karpathy's educational video:  
**"The spelled-out intro to neural networks and backpropagation: building micrograd"**
